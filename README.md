# English-Premier-League-Top-4-Prediction

Constructed a predictive model estimating whether which teams would make top 4 in the league using data from previous years. Machine Learning algorithms such as Random Forest Classifier, Linear Regression, logiestic regression and Gaussian Naive Bayes were employed. 

Correlation analysis was performed on data of Tottenham HotSpur. Investigated how different statistical parameters in soccer affected the result of the each game. 

Sorted out teams that did better or worse in scoring compared to the amount of money teams invested in offenders.
